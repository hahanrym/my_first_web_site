태그 : 설명

앞에 있는 태그 ‘열리는 태그’ , 뒤에 있는 태그는 ‘닫히는 태그’
앞에 있는 것과 뒤에 있는 것을 구분 하기 위해서 ‘ / ‘ 붙임

strong ... /strong 진하게 표시
u ... /u 밑 줄 underline
h1 ... /h1 제목 태그 (숫자가 클수록 문자가 작아짐)
h2 .../h2
h3 .../h3
h4 .../h4
h5 .../h5
h6 .../h6
br 줄바꿈 태그 rbr 간격이 더 커짐
p ... /p 락을 나타내는 태그
img 사진 태그 img src=”사진 이름” "width=숫자“ width -> 사진의 크기
li ... /li 목차 태그 list
ul ... /ul 순서가 없는 목록
ol ... /ol 순서가 있는 목록 자동으로 순서(숫자를) 입력

가장 많이 사용하는 태그

title ... /title 웹페이즈의 제목 태그
meta charset=“utf-8” 본문이 utf-8로 저장해라
body ... /body 본문을 묶는 태그 무조건 약속
head ... /head 바디를 묶는 태그 무조건 약속
html ... /html 모든 태그를 묶는 태그
!doctype html html 태그 위에 관용적으로 쓰는 태그
a ... /a 링크 태그
a href=“주소” 링크 거는 법
a href=“주소” target=“_blank” 새 탭으로 열기
a href=“주소” target=“_blank” title = “제목” 툴팁
