Programming


프로그래밍을 하는 이유는 자동화 된 처리를 하기 위해서. 즉, 누가 언제 사용하든 직관적으로 이해할 수 있고 정보를 처리하는데 걸리는 시간을 단축시키고 정보를 처리하는 작업을 단순화 시키기 위함.

프로그램 : 시간의 순서에 따라 어떤 일이 일어나는 것.
프로그래밍 : 컴퓨터로 하여금 어떠한 일을 순차적으로 실행하도록 하는 것.

프론트엔드 = 클라이언트 사이드 (우리가 보는 화면 처리)
백엔드 = 서버사이트 (DB 조회 등 로직 처리, 화면에 보여줄 정보를 저장, 처리 ,가공하는 부분)

Front-end

* HTML -> 기본적인 웹의 뼈대를 작성하는 작업, 컨텐츠를 쓰는 언어, 마크업 언어 : 구조만 표시하기 위한 언어
* CSS -> 물감, 컨텐츠를 디자인 하는 언어, 마크업 언어
* Javascript -> 컨텐츠를 사용자와 상호작용 할 수 있게 만드는 언어, 움직이게 만드는 언어
